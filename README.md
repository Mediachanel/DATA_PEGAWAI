# Sistem Informasi Data Pegawai (SIKEPEG)

Ringkasan proyek dan petunjuk penggunaan lengkap ada di `DOCUMENTATION.md`.

Isi cepat:
- Frontend: halaman di folder index-based (`/home/`, `/dashboard/`, `/data-pegawai/`, dll) dengan `sidebar.html`/`header.html`/`footer.html`.
- Backend utama: Apps Script (`code.js`) dengan endpoint action-based.
- Opsi backend tambahan: Node/Express (`server.js`) untuk service account.
